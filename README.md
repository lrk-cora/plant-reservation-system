# 植愈空间 - 植物预约管理系统
## 项目简介
本项目是基于 Node.js + Express 开发的全栈植物预约网站，实现前台用户预约、后台管理员动态管理植物与预约数据，采用 JSON 文件轻量化存储数据。

### 功能模块
1. **前台用户端**
- 首页展示热门植物、线上观植/线下种植两大服务
- 全部植物页面，支持分类筛选植物
- 预约下单、查看个人预约记录

2. **后台管理员端**
- 密码登录后台，默认密码：admin123
- 用户预约管理：查看、删除全部预约订单
- 植物动态管理：在线新增、删除植物，上架/下架植物状态，无需修改前端页面

## 技术栈
后端：Node.js、Express
前端：HTML、CSS、原生JavaScript
数据存储：本地 data.json 文件

## 项目运行步骤
1. 打开项目终端，安装依赖
```bash
npm install
```

2. 启动后端服务
```bash
node app.js
```
3. 浏览器访问
```bash
前台首页：http://localhost:3000
管理员后台：http://localhost:3000/html/admin.html
```

## 目录结构
- html：存放所有前端页面
- css：全局样式文件
- js：公共脚本文件
- images：植物图片资源文件夹
- app.js：后端主程序入口
- data.json：自动生成，存放所有业务数据
- node_modules：项目依赖包
